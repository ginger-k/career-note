jQuery对象: 就是通过jQuery包装DOM对象后产生的对象
	    可以使用jQuery里的方法


jQuery对象无法使用DOM对象属性方法
DOM对象也无法使用jQuery对象属性方法

开发规范约定：jQuery引用变量名称前$
	var $variable = jQuery对象;
	var variable = DOM对象;

dom对象转为jQuery对象:
	用$()将DOM对象包装起来
	var variable = document.getElementById("mydiv"); // DOM对象
	var $variable = $(variable);  // jQuery对象

jQuery对象转为dom对象:
	通过 [index] 或 get(index)
	var $variable = $("#mydiv"); // jQuery对象
	var variable = $variable[0]; // DOM对象
	var variable = $variable.get(0); // DOM对象

jQuery对象实际上就是DOM对象的一个集合
jQuery对象可以通过 . 延续: $(".name").html("kobe").css("color":"red");
当前对象: $(this)
